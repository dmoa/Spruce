# Spruce

Spruce is a program that lets you generate simple and concise presentations, as all presentations should be.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
pip install spruce
```

## Usage

```bash
spruce yourfile.txt
```
